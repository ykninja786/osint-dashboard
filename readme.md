# OSINT Dashboard

Educational crawler + Flask UI with Tor integration, SQLite storage, and ethical guardrails.

## Features
- Tor connection status indicator (green/red)
- Metadata capture (URLs, titles, status codes, timestamps)
- Flask dashboard for visualization
- Automated setup with `requirements.txt`

## Installation
```bash
git clone https://github.com/yourusername/osint-dashboard
cd osint-dashboard
pip install -r requirements.txt
