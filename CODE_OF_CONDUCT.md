# Code of Conduct

## Purpose
This project exists to support **educational and professional OSINT research** with a humanitarian focus.  
It must never be used to access, store, or distribute illegal content such as CSAM.

## Our Standards
Contributors are expected to:
- Uphold ethical and legal standards in all contributions.
- Respect human rights and avoid harmful or exploitative practices.
- Maintain transparency and reproducibility in workflows.
- Communicate respectfully and constructively with other contributors.

## Unacceptable Behavior
Examples of unacceptable behavior include:
- Using this project for illegal or exploitative purposes.
- Uploading or linking to harmful or prohibited content.
- Harassment, discrimination, or disrespectful communication.
- Submitting code that intentionally undermines security or ethics.

## Enforcement
- Project maintainers may remove, edit, or reject contributions that violate this Code of Conduct.
- Violations may result in temporary or permanent bans from contributing.

## Attribution
This Code of Conduct is adapted from the Contributor Covenant (v2.1) and tailored for humanitarian OSINT work.
